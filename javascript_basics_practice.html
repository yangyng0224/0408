JavaScript 基础知识练习

一、变量的定义

var、let、const 的区别：
  • var：函数作用域，存在变量提升
  • let：块级作用域，不能重复声明
  • const：块级作用域，声明时必须赋值，不能重新赋值（但对象属性可以修改）

二、变量类型

基本数据类型：
  • Number（数值）、String（字符串）、Boolean（布尔）
  • Undefined（未定义）、Null（空值）
  • Symbol（符号）、BigInt（大整数）
引用数据类型：
  • Object（对象）、Array（数组）、Function（函数）

三、表达式

算术表达式、比较表达式、逻辑表达式： 
// 算术表达式
10 + 5    // 加法
10 - 3    // 减法
4 * 2     // 乘法
10 / 3    // 除法
10 % 3    // 取余

// 比较表达式
5 > 3     // 大于
10 === 10 // 严格相等
'5' == 5  // 宽松相等（不推荐）

// 逻辑表达式
true && false  // 逻辑与
true || false  // 逻辑或
!true           // 逻辑非

四、流程控制

条件语句：if...else、switch

循环语句：for、while、do...while、for...of、for...in

五、函数

函数声明、函数表达式、箭头函数： 
// 函数声明
function add(a, b) {
    return a + b;
}

// 函数表达式
const multiply = function(a, b) {
    return a * b;
};

// 箭头函数
const divide = (a, b) => a / b;

// 默认参数
const greet = (name = "访客") => `你好，${name}！`;
